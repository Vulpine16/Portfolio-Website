# Portfolio-Website
Developing a portfolio website for future employment and employers.

TECHNOLOGIES/SUPPLIES:
I utilized HTML, CSS for styling, and JavaScript for interactivity.

METHOD:
The idea was to keep it minimalist and simple without getting too complex as I didn't have a lot of content to populate it with. The idea was to recreate this semi-psychedelic effect on the text to create this color bleed effect. With RGB colors bouncing off the text that follows the mouse cursor around.

WHAT I LEARNED:
I learned to think a lot more dynamically about what JavaScript is used for and what it can do. And that the more a tool is used the more you'll learn about what it's used for; a language model to create elements that are user-interactive.
